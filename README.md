# 📊 Global YouTube Statistics Analysis

## 🚀 Project Overview

This project performs an in-depth Exploratory Data Analysis (EDA) on a Global YouTube Statistics dataset using Python and popular data analysis libraries.

The objective is to uncover meaningful insights about YouTube creators, subscribers, video views, content categories, and country-wise trends through statistical analysis and data visualization.

---

## 🎯 Objectives

* Analyze global YouTube channel statistics
* Perform statistical analysis on channel metrics
* Identify top-performing creators and categories
* Explore subscriber and view count distributions
* Discover relationships between key variables
* Generate visual insights using charts and graphs

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains information such as:

* Channel Name
* Category
* Subscribers
* Video Views
* Number of Uploads
* Country
* Channel Rank
* Video Count
* Other performance metrics

---

## 📈 Exploratory Data Analysis

### Data Preprocessing

* Missing value detection
* Data cleaning
* Data type conversion
* Duplicate record checking

### Statistical Analysis

* Mean
* Median
* Mode
* Variance
* Standard Deviation
* Correlation Analysis

### Visualizations

* Histograms
* Bar Charts
* Scatter Plots
* Correlation Heatmaps
* Distribution Analysis
* Country-wise Comparisons

---

## 🔍 Key Insights

* Entertainment channels dominate subscriber rankings.
* Higher video views generally correlate with higher subscriber counts.
* Certain countries contribute significantly more top creators.
* Upload frequency alone does not guarantee channel growth.
* Strong relationships exist between views, subscribers, and engagement metrics.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sahil1000725/global-youtube-statistics-analysis.git
```

Navigate to the project directory:

```bash
cd global-youtube-statistics-analysis
```

Install required libraries:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
